# Demo with Playwright

## Step to run

Clone project
```
$git clone
$cd demo-playwright
$npm install
```

Run test with headless mode
```
$npx playwright test
```

Run test with headed mode
```
$npx playwright test --headed
```

Open report
```
$npx playwright show-report
```

Run test with UI mode
```
$npx playwright test --ui
```
